For the Arduino, simply flash the arduino code in the Arduino folder onto the Arduino board. Make sure to set the wi-fi connection name and password in the code though...

For the server, it is written in node js, to run locally simply enter node server.js in the folder. Make sure to have all the node dependencies installed before hand (found in package.json).

It is meant to be hosted on heroku, the current url is www.epa4012.herokuapp.com.
